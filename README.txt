БД 1С уже выгружена на сервер. 1С облако. При авторизации пароли не требуется (Двухфакторная авторизация отключена). Если высвечивается всплывающее окно, то нужно установить все галочки и нажать кнопку "Выполнить запуск". Это связано с ограничением количества пользователей в пробном аккаунте облаке. Можно выгрузить на локальный сервер (APACHE хорошо работает). Для этих случаев существует по копии на каждый вариант бота. Для облачного подключения используется файл с указанным уточнением, а оставшиеся будут обращаться к локальному серверу и не будет работать, если его нет. В файлах *.py хранится код самого бота, а файлы *.exe являются собранными файлами, и не требуют установленных библиотек. Это упрощает запуск, потому что должно запуститься на любом компьютере.

На главной странице меню выбора услуги, а по кнопке "скрыть диаграмму" ("показать диаграмму") можно просмотреть столбчатую диаграмму с указанием всех незакрытых талонов с разрезом по услугам. В списке талонов можно просмотреть все талоны (цвета строк зависят от статуса талона) и выбрать любой из них. По кнопке "взять в работу" можно установить статус обработки для талона. По кнопке закрыть талон нужно выбрать оператора, который закрывает талон. Отбор идёт только по тем, которые могут выполнять выбранную услугу. Список этих услуг редактируется для каждого оператора отдельно. Необходимо через панель администратора зайти в список операторов, затем выбрать существующего оператора, или создать нового, а затем перейти по гиперссылке "услуги операторов" и пополнить список услуг путем добавления новых строк и выбора услуги из предложенного списка. После этого данный оператор сможет закрывать талон по данной услуге. Можно воспользоваться кнопкой "взять следующий", и система сама выберет следующий талон с учётом указанного времени и приоритету талона. По умолчанию устанавливается в значение приоритета услуги, но может изменяться. Приоритеты услуг настраиваются администратором прямо в списке возможных услуг. Если пользоваться главным меню приложения, то можно сразу после ввода времени вывести талон на печать, или же сохранить его в один из множества форматов файла. Через телеграм-бота талон приходит в виде электронного *.pdf файла. При взятии следующего талона, только на локальной версии можно реализовать озвучивание номера талона. Для этого нужно распаковать архив audio и поместить его на диск "D". В последствии этот пункт не будет являться необходимым и файл будет запускаться без особых дополнительных требований. Это опять-таки связано с веб-клиентом и сопутствующими ему проблемами. Через телеграм-бота можно брать любой талон, но существует лимит этих действий. Количество возможных талонов указывается администратором (в выпадающем списке "сервис") и сбрасывается каждый день. Инструкция бота "Выберите наступающую дату", означает, что скорее всего вы ввели ту дату, которая уже прошла. Для того, чтобы взять эл. талон нужно быть зарегистрированным в боте. Выполняется через главное меню по одноразовому уникальному ключу. Команда "Регистрация в телеграм".
